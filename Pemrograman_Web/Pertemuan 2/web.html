Berkenalan Dengan Pemrograman Web

Web Programmingatau Pemrograman Web merupakan istilah yang eratg kaitannya dengan internet dan website. Memang benar, karena pemrograman web merupakan suatu proses pembuatan website untuk keperluan internet. Orang banyak mengenal web dengan istikah WWW atau World Wide Web


World Wide Web adalah halaman-halaman website yang terkoneksi satu dengan yang lainnya atau disebut juga dengan hyperlink yang membentuk samudera informasiyang berjalan dengan protokol Hypertext Tranfer Protokol (HTTP)

Kelebihan Aplikasi Berbasis Web:

  1. Mudah diakses dan dapat diakses diberbagaijenis perangkat seperti, perangkat Mobile/Smartphone, perangkat dekstop, dan perangkat tablet.
  2. Dapat dijalankan diberbagai macam sistem informasi (OS) seperti Mac, Windows, Linux, dll.
  3. Dapat diakses dari mana saja dan kapan saja, asalkan terhubung dengan jaringan intenret maupun jaringan lokal tempat aplikasi Berbasis Web itu berada.

Kekurangan Aplikasi Berbasis Web

  • Memerlukan koneksi internet yang stabil agar tetap dapat menggunakan aplikasinya secara nyaman.
  • Memerlukan sistem keamanan yang baik, baik dari sisi server, website, koneksi yang digunakan, dll.
